Glass: быстрый гайд для друга (Windows)

1) Установить зависимости:
- Node.js 20.x
- Python 3.x
- Visual Studio Build Tools

2) В папке проекта:
- npm run setup

3) Запуск:
- обычный: npm start
- скрытый: двойной клик Start-Glass-Hidden.vbs

4) Если не стартует:
- открыть start-glass.log
- проверить node -v (должно быть 20.x)
