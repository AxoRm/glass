# Скрытый запуск Glass (Windows)

Этот гайд для запуска без консольного окна и без лишних debug-окон.

## 1) Подготовка

1. Установите Node.js 20.x, Python 3.x и Visual Studio Build Tools.
2. В корне проекта выполните:

```bash
npm run setup
```

## 2) Способы запуска

Обычный запуск (для проверки ошибок):

```bash
npm start
```

Скрытый запуск:

- Двойной клик по `Start-Glass-Hidden.vbs`

Скрипт пробует запуск по приоритету:

1. `dist\\Glass Portable.exe`
2. `dist\\win-unpacked\\Glass.exe`
3. Фоновый `npm start` (если сборки нет)

## 3) Где смотреть логи

- Основной лог скрытого старта: `start-glass.log`
- Доп. логи приложения (если созданы): `run.log`, `run2.log`, `run-new.log`

Если при двойном клике "ничего не происходит", сначала откройте `start-glass.log`.

## 4) Автозапуск вместе с Windows

1. Нажмите `Win + R`
2. Введите `shell:startup`
3. Скопируйте туда `Start-Glass-Hidden.vbs` (или ярлык на него)

После этого приложение будет подниматься скрытно при входе в систему.

## 5) Быстрые проверки при проблемах

1. `node -v` должен быть `20.x`.
2. Запустить вручную `npm start` и убедиться, что GUI открывается.
3. Проверить, что антивирус/Defender не блокирует `wscript.exe` или Electron exe.
